# KOTO — Elevated Essentials

A premium fashion e-commerce website for a Cairo-based clothing brand. Built as a portfolio project demonstrating modern front-end development and UI/UX design.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll animations and reveal effects
- Product filtering by category (All / Tops / Bottoms / Outerwear)
- Add-to-cart interactions with count indicator and toast notifications
- Mobile navigation with animated hamburger menu
- Newsletter subscription with validation
- Marquee ticker, hero animations, and scroll indicator

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (no frameworks)
- Google Fonts: Cormorant Garant + Inter
